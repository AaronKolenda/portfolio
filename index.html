Projects

This is my final project for the Iron Yard. It is a drum machine I built with Javascript and Backbone.js, with a simple Node.js back end. The user can create and save a new drum beat, or edit an already existing beat or demo beat. Each beat is saved by creating a uniquely generated url to access it.

This is a Choose Your Own Adventure-style book editor. The user can create pages in book, with action-based links to other book pages ("if you want to do X, turn to page Y"). The user can also edit pages, delete pages, and view the table of contents. This project is meant to use every feature of the Backbone.js framework, including models, routers, views, sync, and collections.

The purpose of this project was to practice recreating a pixel perfect html/css mock up from a psd. The psd was made by an actual designer, and can be seen here for comparison.

This is a clock that changes the color of the background depending on what time it is. The time is converted into a hexidecimal value. The user can click to toggle back and forth between the current time and current color.

This is a recreation of the main section of the old ESPN homepage. The point of this project was to use Angular.js to get the data via an api, and display it on the page. Using css, I've recreated the exact look of the original website, and even made it more responsive than the actual old ESPN site. They've updated thier site since then, but here is an old screenshot for comparison.
