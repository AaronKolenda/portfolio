PROJECTS

Drum Chum was my final project for the Iron Yard. It is a drum machine I built with Javascript and Backbone.js, with a simple Node.js back end. The user can create and save a new drum beat, or edit an already existing beat or demo beat. Each beat is saved by creating a uniquely generated url to access it.

Write Your Own Adventure is a choose your own adventure-style book editor. The user can create pages in book, with action-based links to other book pages ("if you want to do X, turn to page Y"). The user can also edit pages, delete pages, and view the table of contents. This project is meant to use every feature of the Backbone.js framework, including models, routers, views, sync, and collections.
